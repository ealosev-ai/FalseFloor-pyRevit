# Raised Floor Parameter Reference

## 1. Управляющий объект: перекрытие-зона `RaisedFloor_Zone`

Главный объект, на котором хранится вся логика зоны, настройки генерации и связи с результатами.

### 1.1. Идентификация и статус

| Параметр              | Тип данных | Уровень   | Где хранится | Назначение                                                      |
| --------------------- | ---------- | --------- | ------------ | --------------------------------------------------------------- |
| `RF_Zone_Code`         | Текст      | Экземпляр | Перекрытие   | Уникальный код зоны                                             |
| `RF_Gen_Status` | Текст      | Экземпляр | Перекрытие   | Служебный статус: подготовлено / сетка / плитки / каркас и т.д. |

### 1.2. Геометрия и сетка

| Параметр              | Тип данных | Уровень   | Где хранится | Назначение                      |
| --------------------- | ---------- | --------- | ------------ | ------------------------------- |
| `RF_Step_X`            | Длина      | Экземпляр | Перекрытие   | Шаг сетки по X                  |
| `RF_Step_Y`            | Длина      | Экземпляр | Перекрытие   | Шаг сетки по Y                  |
| `RF_Offset_X`       | Длина      | Экземпляр | Перекрытие   | Подобранное смещение сетки по X |
| `RF_Offset_Y`       | Длина      | Экземпляр | Перекрытие   | Подобранное смещение сетки по Y |
| `RF_Base_X`           | Длина      | Экземпляр | Перекрытие   | Базовая координата X            |
| `RF_Base_Y`           | Длина      | Экземпляр | Перекрытие   | Базовая координата Y            |
| `RF_Base_Z`           | Длина      | Экземпляр | Перекрытие   | Базовая отметка Z               |
| `RF_Floor_Height` | Длина      | Экземпляр | Перекрытие   | Общая высота фальшпола          |

### 1.3. Контур и служебные связи

| Параметр             | Тип данных | Уровень   | Где хранится | Назначение                        |
| -------------------- | ---------- | --------- | ------------ | --------------------------------- |
| `RF_Contour_Lines_ID` | Текст      | Экземпляр | Перекрытие   | ID линий контура, разделитель `;` |
| `RF_Grid_Lines_ID`   | Текст      | Экземпляр | Перекрытие   | ID линий сетки, разделитель `;`   |
| `RF_Base_Marker_ID`  | Текст      | Экземпляр | Перекрытие   | ID крестика базовой точки         |

### 1.4. Результаты генерации

| Параметр                | Тип данных | Уровень   | Где хранится | Назначение            |
| ----------------------- | ---------- | --------- | ------------ | --------------------- |
| `RF_Tiles_ID`          | Текст      | Экземпляр | Перекрытие   | ID размещённых плиток |
| `RF_Stringers_Top_ID` | Текст      | Экземпляр | Перекрытие   | ID верхних лонжеронов |
| `RF_Stringers_Bottom_ID`  | Текст      | Экземпляр | Перекрытие   | ID нижних лонжеронов  |
| `RF_Supports_ID`           | Текст      | Экземпляр | Перекрытие   | ID стоек              |

### 1.5. Настройки каркаса

| Параметр                  | Тип данных | Уровень   | Где хранится | Назначение                                                 |
| ------------------------- | ---------- | --------- | ------------ | ---------------------------------------------------------- |
| `RF_Top_Direction`  | Текст      | Экземпляр | Перекрытие   | `X` или `Y`                                                |
| `RF_Bottom_Mode`         | Текст      | Экземпляр | Перекрытие   | Регулярные / Только под стыками / Регулярные + под стыками |
| `RF_Bottom_Step`           | Длина      | Экземпляр | Перекрытие   | Шаг нижних лонжеронов                                      |
| `RF_Max_Stringer_Len` | Длина      | Экземпляр | Перекрытие   | Максимальная длина сегмента                                |
| `RF_Tile_Thickness`       | Длина      | Экземпляр | Перекрытие   | Толщина плитки (записывается скриптом 05)                  |

### 1.6. Опционально (будущее)

| Параметр                   | Тип данных | Уровень   | Где хранится | Назначение            |
| -------------------------- | ---------- | --------- | ------------ | --------------------- |
| `RF_Frame_Type`           | Текст      | Экземпляр | Перекрытие   | Тип схемы каркаса     |
| `RF_Tile_Type_Базовый`    | Текст      | Экземпляр | Перекрытие   | Базовый тип плитки    |
| `RF_Stringer_Type_Базовый` | Текст      | Экземпляр | Перекрытие   | Базовый тип лонжерона |
| `RF_Support_Type_Default`    | Текст      | Экземпляр | Перекрытие   | Базовый тип стойки    |

## 2. Плитка `RF_Tile`

Экземпляр плитки в конкретной ячейке.

### 2.1. Идентификация

| Параметр   | Тип данных | Уровень   | Где хранится | Назначение              |
| ---------- | ---------- | --------- | ------------ | ----------------------- |
| `RF_Mark` | Текст      | Экземпляр | Плитка       | Уникальная марка плитки |
| `RF_Zone`  | Текст      | Экземпляр | Плитка       | Код зоны                |

### 2.2. Положение в сетке

| Параметр     | Тип данных | Уровень   | Где хранится | Назначение     |
| ------------ | ---------- | --------- | ------------ | -------------- |
| `RF_Column` | Целое      | Экземпляр | Плитка       | Индекс столбца |
| `RF_Row`     | Целое      | Экземпляр | Плитка       | Индекс ряда    |

### 2.3. Классификация

| Параметр        | Тип данных | Уровень   | Где хранится | Назначение                  |
| --------------- | ---------- | --------- | ------------ | --------------------------- |
| `RF_Tile_Type` | Текст      | Экземпляр | Плитка       | Full / SimpleCut / ComplexCut |

### 2.4. Геометрия экземпляра

Фактические размеры плитки. Эталонное имя — `RF_Actual_Size_X/Y`.
Допустимо временно использовать `RF_Cut_X/Y` если уже заведены.

| Параметр           | Тип данных | Уровень   | Где хранится | Назначение                  |
| ------------------ | ---------- | --------- | ------------ | --------------------------- |
| `RF_Actual_Size_X` | Длина      | Экземпляр | Плитка       | Реальный размер плитки по X |
| `RF_Actual_Size_Y` | Длина      | Экземпляр | Плитка       | Реальный размер плитки по Y |

### 2.5. Параметры семейства плитки

| Параметр        | Тип данных | Уровень             | Где хранится     | Назначение              |
| --------------- | ---------- | ------------------- | ---------------- | ----------------------- |
| `RF_Thickness`    | Длина      | Тип                 | Семейство плитки | Толщина плитки          |
| `RF_Size_X`   | Длина      | Тип                 | Семейство плитки | Номинальный размер по X |
| `RF_Size_Y`   | Длина      | Тип                 | Семейство плитки | Номинальный размер по Y |
| `RF_Real_X` | Длина      | Экземпляр / формула | Семейство плитки | Реальный размер по X    |
| `RF_Real_Y` | Длина      | Экземпляр / формула | Семейство плитки | Реальный размер по Y    |

**Формулы** (при использовании старой схемы `RF_Cut_X/Y`):

| Параметр        | Формула                                             |
| --------------- | --------------------------------------------------- |
| `RF_Real_X` | `if(RF_Cut_X > 0, RF_Cut_X, RF_Size_X)` |
| `RF_Real_Y` | `if(RF_Cut_Y > 0, RF_Cut_Y, RF_Size_Y)` |

При переходе на `RF_Actual_Size_X/Y` формулы не нужны — геометрию сразу вешать на эти параметры.

## 3. Лонжерон `RF_Stringer`

Одно семейство, два типоразмера: верхний (плашмя) и нижний (на ребро).
Геометрия строится от нижней опорной плоскости — низ профиля на Reference Plane.

Типы (пример):

- `Верх_80x40` — ширина 80, высота 40
- `Низ_40x80` — ширина 40, высота 80

### 3.1. Идентификация

| Параметр   | Тип данных | Уровень   | Где хранится | Назначение                                  |
| ---------- | ---------- | --------- | ------------ | ------------------------------------------- |
| `RF_Mark` | Текст      | Экземпляр | Лонжерон     | Уникальная марка (ВЛ.1.2400мм / НЛ.3.600мм) |
| `RF_Zone`  | Текст      | Экземпляр | Лонжерон     | Код зоны                                    |

### 3.2. Классификация

| Параметр             | Тип данных | Уровень   | Где хранится | Назначение       |
| -------------------- | ---------- | --------- | ------------ | ---------------- |
| `RF_Stringer_Type`   | Текст      | Экземпляр | Лонжерон     | Upper / Lower |
| `RF_Direction_Axis` | Текст      | Экземпляр | Лонжерон     | X / Y            |

### 3.3. Спецификация профиля

| Параметр             | Тип данных | Уровень | Где хранится        | Назначение      |
| -------------------- | ---------- | ------- | ------------------- | --------------- |
| `RF_Type_Name`      | Текст      | Тип     | Семейство лонжерона | Имя типа        |
| `RF_Profile_Width`  | Длина      | Тип     | Семейство лонжерона | Ширина профиля  |
| `RF_Profile_Height`  | Длина      | Тип     | Семейство лонжерона | Высота профиля  |
| `RF_Wall_Thickness` | Длина      | Тип     | Семейство лонжерона | Толщина металла |

### 3.4. Геометрия

| Параметр | Тип данных | Уровень   | Где хранится | Назначение                |
| -------- | ---------- | --------- | ------------ | ------------------------- |
| `Length` | Системный  | Экземпляр | Revit        | Длина line-based элемента |

### 3.5. Опционально

| Параметр             | Тип данных | Уровень   | Где хранится | Назначение                          |
| -------------------- | ---------- | --------- | ------------ | ----------------------------------- |
| `RF_Frame_Level` | Текст      | Экземпляр | Лонжерон     | Например Upper layer / Lower layer |

## 4. Стойка `RF_Support`

Отдельное семейство стойки.

### 4.1. Идентификация

| Параметр   | Тип данных | Уровень   | Где хранится | Назначение              |
| ---------- | ---------- | --------- | ------------ | ----------------------- |
| `RF_Mark` | Текст      | Экземпляр | Стойка       | Уникальная марка (СТ.1) |
| `RF_Zone`  | Текст      | Экземпляр | Стойка       | Код зоны                |

### 4.2. Привязка к сетке

| Параметр     | Тип данных | Уровень   | Где хранится | Назначение          |
| ------------ | ---------- | --------- | ------------ | ------------------- |
| `RF_Column` | Целое      | Экземпляр | Стойка       | Индекс столбца узла |
| `RF_Row`     | Целое      | Экземпляр | Стойка       | Индекс ряда узла    |

### 4.3. Спецификация

| Параметр             | Тип данных | Уровень           | Где хранится     | Назначение               |
| -------------------- | ---------- | ----------------- | ---------------- | ------------------------ |
| `RF_Type_Name`      | Текст      | Тип               | Семейство стойки | Имя типа                 |
| `RF_Support_Height`   | Длина      | Тип или экземпляр | Семейство стойки | Высота стойки            |
| `RF_Base_Size`    | Длина      | Тип               | Семейство стойки | Размер нижней опоры      |
| `RF_Head_Size` | Длина      | Тип               | Семейство стойки | Размер верхнего оголовка |

### 4.4. Опционально

| Параметр        | Тип данных | Уровень   | Где хранится | Назначение                      |
| --------------- | ---------- | --------- | ------------ | ------------------------------- |
| `RF_Support_Mode` | Текст      | Экземпляр | Стойка       | Если будут разные режимы стойки |
