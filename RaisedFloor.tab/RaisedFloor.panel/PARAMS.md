# Эталонная структура параметров — Фальшпол

## 1. Управляющий объект: перекрытие-зона `Фальшпол_Зона`

Главный объект, на котором хранится вся логика зоны, настройки генерации и связи с результатами.

### 1.1. Идентификация и статус

| Параметр              | Тип данных | Уровень   | Где хранится | Назначение                                                      |
| --------------------- | ---------- | --------- | ------------ | --------------------------------------------------------------- |
| `FP_Код_Зоны`         | Текст      | Экземпляр | Перекрытие   | Уникальный код зоны                                             |
| `FP_Статус_Генерации` | Текст      | Экземпляр | Перекрытие   | Служебный статус: подготовлено / сетка / плитки / каркас и т.д. |

### 1.2. Геометрия и сетка

| Параметр              | Тип данных | Уровень   | Где хранится | Назначение                      |
| --------------------- | ---------- | --------- | ------------ | ------------------------------- |
| `FP_Шаг_X`            | Длина      | Экземпляр | Перекрытие   | Шаг сетки по X                  |
| `FP_Шаг_Y`            | Длина      | Экземпляр | Перекрытие   | Шаг сетки по Y                  |
| `FP_Смещение_X`       | Длина      | Экземпляр | Перекрытие   | Подобранное смещение сетки по X |
| `FP_Смещение_Y`       | Длина      | Экземпляр | Перекрытие   | Подобранное смещение сетки по Y |
| `FP_База_X`           | Длина      | Экземпляр | Перекрытие   | Базовая координата X            |
| `FP_База_Y`           | Длина      | Экземпляр | Перекрытие   | Базовая координата Y            |
| `FP_База_Z`           | Длина      | Экземпляр | Перекрытие   | Базовая отметка Z               |
| `FP_Высота_Фальшпола` | Длина      | Экземпляр | Перекрытие   | Общая высота фальшпола          |

### 1.3. Контур и служебные связи

| Параметр             | Тип данных | Уровень   | Где хранится | Назначение                        |
| -------------------- | ---------- | --------- | ------------ | --------------------------------- |
| `FP_ID_ЛинийКонтура` | Текст      | Экземпляр | Перекрытие   | ID линий контура, разделитель `;` |
| `FP_ID_ЛинийСетки`   | Текст      | Экземпляр | Перекрытие   | ID линий сетки, разделитель `;`   |
| `FP_ID_МаркераБазы`  | Текст      | Экземпляр | Перекрытие   | ID крестика базовой точки         |

### 1.4. Результаты генерации

| Параметр                | Тип данных | Уровень   | Где хранится | Назначение            |
| ----------------------- | ---------- | --------- | ------------ | --------------------- |
| `FP_ID_Плиток`          | Текст      | Экземпляр | Перекрытие   | ID размещённых плиток |
| `FP_ID_Лонжеронов_Верх` | Текст      | Экземпляр | Перекрытие   | ID верхних лонжеронов |
| `FP_ID_Лонжеронов_Низ`  | Текст      | Экземпляр | Перекрытие   | ID нижних лонжеронов  |
| `FP_ID_Стоек`           | Текст      | Экземпляр | Перекрытие   | ID стоек              |

### 1.5. Настройки каркаса

| Параметр                  | Тип данных | Уровень   | Где хранится | Назначение                                                 |
| ------------------------- | ---------- | --------- | ------------ | ---------------------------------------------------------- |
| `FP_Направление_Верхних`  | Текст      | Экземпляр | Перекрытие   | `X` или `Y`                                                |
| `FP_Режим_Нижних`         | Текст      | Экземпляр | Перекрытие   | Регулярные / Только под стыками / Регулярные + под стыками |
| `FP_Шаг_Нижних`           | Длина      | Экземпляр | Перекрытие   | Шаг нижних лонжеронов                                      |
| `FP_Макс_Длина_Лонжерона` | Длина      | Экземпляр | Перекрытие   | Максимальная длина сегмента                                |
| `FP_Толщина_Плитки`       | Длина      | Экземпляр | Перекрытие   | Толщина плитки (записывается скриптом 05)                  |

### 1.6. Опционально (будущее)

| Параметр                   | Тип данных | Уровень   | Где хранится | Назначение            |
| -------------------------- | ---------- | --------- | ------------ | --------------------- |
| `FP_Тип_Каркаса`           | Текст      | Экземпляр | Перекрытие   | Тип схемы каркаса     |
| `FP_Тип_Плитки_Базовый`    | Текст      | Экземпляр | Перекрытие   | Базовый тип плитки    |
| `FP_Тип_Лонжерона_Базовый` | Текст      | Экземпляр | Перекрытие   | Базовый тип лонжерона |
| `FP_Тип_Стойки_Базовый`    | Текст      | Экземпляр | Перекрытие   | Базовый тип стойки    |

## 2. Плитка `ФП_Плитка`

Экземпляр плитки в конкретной ячейке.

### 2.1. Идентификация

| Параметр   | Тип данных | Уровень   | Где хранится | Назначение              |
| ---------- | ---------- | --------- | ------------ | ----------------------- |
| `FP_Марка` | Текст      | Экземпляр | Плитка       | Уникальная марка плитки |
| `FP_Зона`  | Текст      | Экземпляр | Плитка       | Код зоны                |

### 2.2. Положение в сетке

| Параметр     | Тип данных | Уровень   | Где хранится | Назначение     |
| ------------ | ---------- | --------- | ------------ | -------------- |
| `FP_Колонка` | Целое      | Экземпляр | Плитка       | Индекс столбца |
| `FP_Ряд`     | Целое      | Экземпляр | Плитка       | Индекс ряда    |

### 2.3. Классификация

| Параметр        | Тип данных | Уровень   | Где хранится | Назначение                  |
| --------------- | ---------- | --------- | ------------ | --------------------------- |
| `FP_Тип_Плитки` | Текст      | Экземпляр | Плитка       | Полная / Подрезка / Сложная |

### 2.4. Геометрия экземпляра

Фактические размеры плитки. Эталонное имя — `FP_Размер_Факт_X/Y`.
Допустимо временно использовать `FP_Подрезка_X/Y` если уже заведены.

| Параметр           | Тип данных | Уровень   | Где хранится | Назначение                  |
| ------------------ | ---------- | --------- | ------------ | --------------------------- |
| `FP_Размер_Факт_X` | Длина      | Экземпляр | Плитка       | Реальный размер плитки по X |
| `FP_Размер_Факт_Y` | Длина      | Экземпляр | Плитка       | Реальный размер плитки по Y |

### 2.5. Параметры семейства плитки

| Параметр        | Тип данных | Уровень             | Где хранится     | Назначение              |
| --------------- | ---------- | ------------------- | ---------------- | ----------------------- |
| `FP_Толщина`    | Длина      | Тип                 | Семейство плитки | Толщина плитки          |
| `FP_Размер_X`   | Длина      | Тип                 | Семейство плитки | Номинальный размер по X |
| `FP_Размер_Y`   | Длина      | Тип                 | Семейство плитки | Номинальный размер по Y |
| `FP_Реальный_X` | Длина      | Экземпляр / формула | Семейство плитки | Реальный размер по X    |
| `FP_Реальный_Y` | Длина      | Экземпляр / формула | Семейство плитки | Реальный размер по Y    |

**Формулы** (при использовании старой схемы `FP_Подрезка_X/Y`):

| Параметр        | Формула                                             |
| --------------- | --------------------------------------------------- |
| `FP_Реальный_X` | `if(FP_Подрезка_X > 0, FP_Подрезка_X, FP_Размер_X)` |
| `FP_Реальный_Y` | `if(FP_Подрезка_Y > 0, FP_Подрезка_Y, FP_Размер_Y)` |

При переходе на `FP_Размер_Факт_X/Y` формулы не нужны — геометрию сразу вешать на эти параметры.

## 3. Лонжерон `ФП_Лонжерон`

Одно семейство, два типоразмера: верхний (плашмя) и нижний (на ребро).
Геометрия строится от нижней опорной плоскости — низ профиля на Reference Plane.

Типы (пример):

- `Верх_80x40` — ширина 80, высота 40
- `Низ_40x80` — ширина 40, высота 80

### 3.1. Идентификация

| Параметр   | Тип данных | Уровень   | Где хранится | Назначение                                  |
| ---------- | ---------- | --------- | ------------ | ------------------------------------------- |
| `FP_Марка` | Текст      | Экземпляр | Лонжерон     | Уникальная марка (ВЛ.1.2400мм / НЛ.3.600мм) |
| `FP_Зона`  | Текст      | Экземпляр | Лонжерон     | Код зоны                                    |

### 3.2. Классификация

| Параметр             | Тип данных | Уровень   | Где хранится | Назначение       |
| -------------------- | ---------- | --------- | ------------ | ---------------- |
| `FP_Тип_Лонжерона`   | Текст      | Экземпляр | Лонжерон     | Верхний / Нижний |
| `FP_Ось_Направления` | Текст      | Экземпляр | Лонжерон     | X / Y            |

### 3.3. Спецификация профиля

| Параметр             | Тип данных | Уровень | Где хранится        | Назначение      |
| -------------------- | ---------- | ------- | ------------------- | --------------- |
| `FP_Типоразмер`      | Текст      | Тип     | Семейство лонжерона | Имя типа        |
| `FP_Ширина_Профиля`  | Длина      | Тип     | Семейство лонжерона | Ширина профиля  |
| `FP_Высота_Профиля`  | Длина      | Тип     | Семейство лонжерона | Высота профиля  |
| `FP_Толщина_Профиля` | Длина      | Тип     | Семейство лонжерона | Толщина металла |

### 3.4. Геометрия

| Параметр | Тип данных | Уровень   | Где хранится | Назначение                |
| -------- | ---------- | --------- | ------------ | ------------------------- |
| `Length` | Системный  | Экземпляр | Revit        | Длина line-based элемента |

### 3.5. Опционально

| Параметр             | Тип данных | Уровень   | Где хранится | Назначение                          |
| -------------------- | ---------- | --------- | ------------ | ----------------------------------- |
| `FP_Уровень_Каркаса` | Текст      | Экземпляр | Лонжерон     | Например Верхний слой / Нижний слой |

## 4. Стойка `ФП_Стойка`

Отдельное семейство стойки.

### 4.1. Идентификация

| Параметр   | Тип данных | Уровень   | Где хранится | Назначение              |
| ---------- | ---------- | --------- | ------------ | ----------------------- |
| `FP_Марка` | Текст      | Экземпляр | Стойка       | Уникальная марка (СТ.1) |
| `FP_Зона`  | Текст      | Экземпляр | Стойка       | Код зоны                |

### 4.2. Привязка к сетке

| Параметр     | Тип данных | Уровень   | Где хранится | Назначение          |
| ------------ | ---------- | --------- | ------------ | ------------------- |
| `FP_Колонка` | Целое      | Экземпляр | Стойка       | Индекс столбца узла |
| `FP_Ряд`     | Целое      | Экземпляр | Стойка       | Индекс ряда узла    |

### 4.3. Спецификация

| Параметр             | Тип данных | Уровень           | Где хранится     | Назначение               |
| -------------------- | ---------- | ----------------- | ---------------- | ------------------------ |
| `FP_Типоразмер`      | Текст      | Тип               | Семейство стойки | Имя типа                 |
| `FP_Высота_Стойки`   | Длина      | Тип или экземпляр | Семейство стойки | Высота стойки            |
| `FP_Размер_Опоры`    | Длина      | Тип               | Семейство стойки | Размер нижней опоры      |
| `FP_Размер_Оголовка` | Длина      | Тип               | Семейство стойки | Размер верхнего оголовка |

### 4.4. Опционально

| Параметр        | Тип данных | Уровень   | Где хранится | Назначение                      |
| --------------- | ---------- | --------- | ------------ | ------------------------------- |
| `FP_Тип_Стойки` | Текст      | Экземпляр | Стойка       | Если будут разные режимы стойки |
